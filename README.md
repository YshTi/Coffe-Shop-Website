# ☕ Coffee Shop Website

This project is a modern, responsive coffee shop website built using **HTML, CSS, and JavaScript**, bundled with **Vite**.

## 📌 Project Overview

The website showcases a coffee shop with multiple pages, interactive UI elements, and a clean design. It is optimized for performance and structured for scalability.

## 👥 Group Project

This project was originally developed as a [**group project**](https://github.com/luxqxqm/Team-Project) in collaboration with other team members.

After the initial development phase, I created a [**personal clone of the repository**](https://github.com/YshTi/Coffe-Shop-Website) to:

- Improve the code structure
- Fix issues and bugs
- Add new features and enhancements
- Experiment with deployment and optimization

> ⚠️ This repository is an **independent continuation** of the original group work.

## 🚀 Features

- Responsive design (mobile-first)
- Multi-page layout
- Modular JavaScript structure
- Asset optimization via Vite
- Dynamic UI components
- Dark/light theme support (favicon adaptation included)

## ⚙️ Installation & Setup

Clone the repository:
```bash
git clone https://github.com/YshTi/Coffe-Shop-Website.git
cd Coffe-Shop-Website
```
Install dependencies:
```bash
npm install
```
Run development server:
```bash
npm run dev
```
Build for production:
```bash
npm run build
```
Preview production build:
```bash
npm run preview
```
